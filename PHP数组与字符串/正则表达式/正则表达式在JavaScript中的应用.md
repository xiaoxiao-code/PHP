# 正则表达式在JavaScript中的应用

# 1 JavaScript 中的`test()`方法

在 JavaScript 中，用于验证用户输入数据是否满足指定格式的最重要的正则表达式函数是`test()` 。
## 1.1 语法格式
`test()`方法是正则表达式对象的一个​​方法，其语法格式如下：
```javascript
正则表达式.test(字符串)
```
## 1.2 功能说明
- `test()`方法会在`字符串`中查找与`正则表达式`相匹配的内容。
- 如果找到了匹配项，则返回`true` 。
- 如果没有找到匹配项，则返回`false` 。
## 1.3 关键区别：是否使用`^`和`$`
`test()`方法的行为在有无`^`（匹配开头）和`$`（匹配结尾）时有显着不同：
- 清晰`^`和`$`： 只要`正则表达式`是`字符串`一个子串，该函数就返回`true` 。
- 包含`^`和`$`： 只有当`正则表达式`与完全匹配时，该函数才返回`字符串` `true` 。
## 1.4 示例
```javascript
<script language="javascript">
    // 模式a: 匹配任意 6 个数字
    var a = /\d{6}/; 
    
    // 模式b: 匹配开头(^)到结尾($) 恰好是 6 个数字
    var b = /^\d{6}$/; 

    // 测试字符串 "12345678"
    document.write(a.test("12345678")); // 返回 true
    document.write(b.test("12345678")); // 返回 false
</script>
```
