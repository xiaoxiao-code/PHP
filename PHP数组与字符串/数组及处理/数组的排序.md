# 数组的排序

# 1 升序排序 (从小到大)

## 1.1 `sort()` 函数
- **格式：** 
```php
sort(数组名)
```
- **功能：** 对数组的**值**进行升序排序 。
- **重要特性：** 此函数会**丢弃**原有的键名，并将数组的键名**修改为从 0 开始的连续整数键名** 。
## 1.2 `asort()` 函数
- **格式：** 
```php
asort(数组名)
```
- **功能：** 对数组的**值**进行升序排序 。
- **重要特性：** 此函数会**保持**数组的键名和值之间的关联 。
示例 ：
```php
<?php
// sort() 示例
$arr1 = array("a" => 5, "x" => 3, 5 => 7, "c" => 1);
sort($arr1);
print_r($arr1);
// 输出: Array ( [0] => 1 [1] => 3 [2] => 5 [3] => 7 )
// (注意：键名 "a", "x", 5, "c" 丢失了)

// asort() 示例
$arr2 = array(2 => "c", 4 => "a", 1 => "b");
asort($arr2);
print_r($arr2);
// 输出: Array ( [4] => a [1] => b [2] => c )
// (注意：键名 4, 1, 2 被保留)
?>
```
# 2 降序排序 (从大到小)
## 2.1 `rsort()` 函数
- **功能：** `sort()` 的降序版本。对数组的值进行降序排序，并**修改**键名为 0 基整数。
## 2.2 `arsort()` 函数
- **功能：** `asort()` 的降序版本。对数组的值进行降序排序，并**保持**键值关联 。
# 3 对多个数组同时排序
## 3.1 `array_multisort()` 函数
- **格式：**
```php
array_multisort(数组名1, 数组名2, ...)
```
- **功能：** 可以一次对多个一维数组排序 。
- **逻辑：**
    1. 首先对**第一个数组**（`数组名1`）的值进行升序排列 。
    2. 其他数组（`数组名2` 等）中值的顺序，会**按照第一个数组排序后的对应顺序**进行排列 。
    3. 所有数组的长度必须相等 。
    示例 ：
```php
<?php
$ar1 = array(3, 5, 2, 4);
$ar2 = array(8, 6, 9, 7);
// 原始对应关系: (3,8), (5,6), (2,9), (4,7)

array_multisort($ar1, $ar2);

print_r($ar1);
// 输出: Array ( [0] => 2 [1] => 3 [2] => 4 [3] => 5 )
print_r($ar2);
// 输出: Array ( [0] => 9 [1] => 8 [2] => 7 [3] => 6 )
// (对应关系: (2,9), (3,8), (4,7), (5,6))
?>
```
- **说明：** 第一个数组中最小的值是 `2`，它对应的第二个数组中的值是 `9`，因此 `9` 成为第二个数组排序后的第一个元素 。
# 4 打乱数组的顺序
## 4.1 `shuffle()` 函数

- **格式：** 
```php
shuffle(数组名)
```
- **功能：** 打乱（随机化）数组的顺序 。
- **重要特性：** 此函数也会**修改**数组的键名，使其变为从 0 开始的连续整数键名 。
 产生 10 个不重复的随机整数 ：
```php
<?php
// 1. 创建一个 1 到 100 的数组
$arr = range(1, 100);

// 2. 打乱数组顺序
shuffle($arr);

// 3. 取出前 10 个
for ($i = 0; $i < 10; $i++) {
    echo $arr[$i] . ",";
}
?>
```
# 5 按相反顺序排序
## 5.1 `array_reverse()` 函数
- **格式：** 
```php
数组名2 = array_reverse(数组名1 [, key])
```
- **功能：** 将 `数组名1` 按相反顺序排序，并生成一个新的 `数组名2` 。
- **`key` 参数（可选）：**
    - `false` (默认)：新数组 `数组名2` 的键名被修改为从 0 开始的整数 。
    - `true`：新数组 `数组名2` 保持原来的键名 。
    示例 ：
```php
<?php
$array = array("a" => 1, 0 => 2, 1 => 3, 2 => 4);

// 默认 (key = false)
$ar1 = array_reverse($array);
print_r($ar1);
// 输出: Array ( [0] => 4 [1] => 3 [2] => 2 [3] => 1 )

// 保留键名 (key = true)
$ar2 = array_reverse($array, true);
print_r($ar2);
// 输出: Array ( [2] => 4 [1] => 3 [0] => 2 [a] => 1 )
?>
```
