# 其他字符串函数

# 1 字符串转化为数组

## 1.1 `explode()` 函数
- **格式：** 
```php
数组名 = explode(分隔符, 字符串)
```
- **功能：** 使用指定的 `分隔符`，将 `字符串` 切割为若干个子串，并将这些子串作为元素存入一个新的数组中 。
示例 ：
```php
<?php
$str = "can you help me";

// 使用空格 " " 作为分隔符
$arr = explode(" ", $str);

print_r($arr);
/*
输出:
Array
(
    [0] => can
    [1] => you
    [2] => help
    [3] => me
)
*/
?>
```
# 2 数组转化为字符串
## 2.1 `implode()` 函数
- **格式：** 
```php
字符串变量 = implode(连接符, 数组名)
```
- **功能：** `explode()` 的逆操作。它使用指定的 `连接符`（也称为“胶水”），将数组中的所有元素连接成一个单一的字符串 。
示例 ：
```php
<?php
$array = array("hello", "how", "are", "you");

// 使用逗号 "," 作为连接符
$str = implode(",", $array);

echo $str;
// 输出: hello,how,are,you
?>
```
# 3 字符串加密函数
## 3.1 `crypt()` 函数
- **功能：** `crypt()` 函数用于对字符串进行**单向加密**，生成一个密文。单向加密意味着密文**不可以**被还原成明文 。这常用于密码的安全存储。
```php
密文 = crypt(字符串)
```
- 对指定字符串加密，生成一个新的密文。每次调用生成的密文是不同的 。
```php
密文 = crypt(字符串[, 密文])
```
- 这种格式用于**验证密码**。它使用 `密文` 中包含的“盐”（salt，加密时使用的随机串），来对 `字符串`（用户输入的密码）进行加密。如果加密后的结果与原 `密文` 相同，则说明密码正确 。
示例（验证密码）：
```php
<?php
$str = "与时俱进"; // 假设这是用户的原始密码

// 1. (注册时) 生成一个密文并存储
$pass = crypt($str);
echo $pass . "<br>"; // $pass 是存储在数据库中的密文

// 2. (登录时) 验证密码
// $str 是用户登录时输入的密码
// $pass 是从数据库取出的密文
if (crypt($str, $pass) == $pass) {
    echo "验证成功!";
}
?>
```
