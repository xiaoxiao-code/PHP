# 常用的字符串函数

字符串是PHP中非常常用的数据类型，PHP提供了大量内置函数来操作字符串

# 1 计算字符串的长度
```php
strlen(字符串)
```
- **功能：** 返回字符串的字节（bytes）长度。
- **注意：** 这个函数对中文字符的处理方式取决于编码 。
    - 在 `charset=GB2312` 编码下，每个汉字为2个字符（字节）。
    - 在 `charset=UTF-8` 编码下，每个汉字为3个字符（字节） 。
```php
mb_strlen(字符串, 编码方式)
```
- **功能：** (mb_strlen 是多字节字符串函数，书中提到它时是为了对比) 它可以根据指定的编码方式来计算字符数量 。
# 2 改变字母大小写
```php
strtolower(字符串)
```
- **功能：** 将字符串中所有的字母都转化为小写 。
```php
strtoupper(字符串)
```
- **功能：** 将字符串中所有的字母都转化为大写 。
# 3 删除字符串的首尾空格
```php
trim(字符串)
```
- **功能：** 删除字符串**首、尾**两端的空格（或其他预定义字符） 。
```php
ltrim(字符串)
```
- **功能：** 删除字符串首部（左侧）的空格 。
```php
rtrim(字符串)
```
- **功能：** 删除字符串尾部（右侧）的空格 。
# 4 字符串查找
```php
strstr(串1, 串2[, 是否串2之前])
```
- **功能：** 在 `串1` 中（区分大小写）查找 `串2` 首次出现的位置 。
- 返回值 ：
    - 如果找到，并且省略第三个参数（或为 `false`），则返回从 `串2` 首次出现的位置开始，直到 `串1` 结尾的子字符串。
    - 如果找到，且第三个参数为 `true`，则返回在 `串1` 中 `串2` 首次出现位置**之前**的那部分子串 。
    - 如果未找到，则返回 `false`。
```php
stristr(串1, 串2[, 是否串2之前])
```
- **功能：** 与 `strstr` 作用类似，但**不区分大小写** 。
示例 ：
```php
<?php
$str = "hello world";
$find = "llo";
$res1 = strstr($str, $find);
$res2 = strstr($str, $find, true);

echo $res1 . ", " . $res2;
// 输出: llo world, he
?>
```
# 5 截取子串
```php
substr(字符串, n, len)
```
- **功能：** 从 `字符串` 的第 `n` 个字符位置开始，截取 `len` 个字符长度的子串 。
- **注意（同 `strlen`）：** 1个汉字为2个(GB2312)或3个(UTF-8)字符长度 。
- 示例 ：
    - `substr("汕头职院", 2, 4)` (GB2312) = ”头职”
    - `substr("汕头职院", 3, 6)` (UTF-8) = ”头职”
# 6 字符串与ASCII码
```php
ord(字符串)
```
- **功能：** 返回字符串中**第一个字符**的ASCII码值 。
```php
chr(n)
```
- **功能：** 返回指定ASCII码值 `n` 对应的字符 。
示例 ：
```php
<?php
echo ord("a"); // 输出: 97
echo chr(97);  // 输出: a
?>
```
# 7 字符串的比较
```php
strcmp(串1, 串2)
```
- **功能：** 比较两个字符串（区分大小写）。
- **重要：** 此函数**不会**将数字字符串转换为数值，而是按字节进行比较 。
- 返回值：
    - 若 `串1` < `串2`，返回 -1。
    - 若 `串1` = `串2`，返回 0。
    - 若 `串1` > `串2`，返回 1。
- 示例 ： `strcmp("123.5", "9.5")` 返回 -1（因为 "1" 小于 "9"）。
# 8 字符串替换
```php
str_replace(子串1, 子串2, 字符串)
```
- **功能：** 在 `字符串` 中查找所有的 `子串1`，并将其全部替换为 `子串2`，最后返回一个新的字符串 。
示例 ：
```php
<?php
$str = "I love you";
$replace = "lucy";
$end = str_replace("you", $replace, $str);

echo $end;
// 输出: I love lucy
?>
```
