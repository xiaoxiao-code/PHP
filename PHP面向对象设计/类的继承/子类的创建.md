# 子类的创建

# 1 什么是子类的创建？

子类的创建是通过`extends`关键字指定，让一个新类（子类）继承一个已有的类（父类）。
- 父类（Parent Class）：也称基类或超类，是被继承的类（例如`A`）。
- 子类（Subclass）：也称派生类，是继承父类的类（例如`B`）。
# 2 继承的语法
```php
class 子类 extends 父类
{
    // 类的属性和方法定义
}
```
# 3 继承的规则
1. 单继承： PHP 只支持单继承，即一个子类只能有一个父类。
2. 继承内容：子类可以继承父类的所有非私有（非私有）的属性和方法。
    - `public`（公有）成员：被子类继承。
    - `protected`（受保护）成员：被子类继承。
    - `private`（不）成员：不能被子类继承。
3. 构造函数：子类会继承父类的构造函数和构造函数。
4. 扩展：子类除了继承父类的成员外，还可以定义自己的新属性和新方法。
# 4 子类继承父类的示例
这个例子展示了`class B`如何继承`class A`。
代码：
```php
<?php
// 1. 定义父类 A
class A
{
    public $a;
    protected static $b = "string2";
    private $c = "string3"; // 私有成员

    function __construct()
    {
        $this->a = "string1";
    }

    public function a_fun()
    {
        $this->a = "string4";
    }
}

// 2. 定义子类 B，使用 extends 继承 A
class B extends A
{
    // 3. 子类 B 自己定义的新属性
    public $x;

    // 4. 子类 B 自己定义的新方法
    public function b_fun()
    {
        // 5. 子类 B 调用从父类 A 继承的方法
        $this->a_fun(); 
        
        // 6. 子类 B 访问从父类 A 继承的 public 属性
        echo $this->a; // 输出: string4
        
        // 7. 子类 B 访问从父类 A 继承的 protected 属性
        echo B::$b; // 输出: string2
        
        // echo $this->c; // 错误！$c 是 A 的 private 成员, B 无法访问
    }
}

// --- 执行 ---
// 8. 创建子类 B 的对象
$obj = new B;

// 9. 访问继承自 A 的构造函数设置的属性
echo $obj->a; // 输出: string1

// 10. 调用子类 B 自己的方法
$obj->b_fun(); 
?>
```
程序说明：
1. `class B extends A`使`B`成为`A`子类。
2. `B`自动获得了`A`的`public $a`、`protected static $b`属性，以及`__construct()`和`a_fun()`方法。`private $c`继承。
3. `$obj = new B;`：创建`B`的对象时，会自动调用继承来`A`的`__construct()`构造函数，因此`$obj->a`被赋值为“string1”。
4. `$obj->b_fun();`：调用`B`自己的方法。
    - 在该方法内部，`$this->a_fun();`成功调用了继承来的`a_fun()`方法，将`$this->a`其值修改为“string4”。
    - `echo $this->a;`输出“string4”。
    - `echo B::$b;`访问了继承来的受保护的静态属性“string2”。
