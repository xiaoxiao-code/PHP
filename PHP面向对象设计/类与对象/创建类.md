# 创建类

# 1 什么是创建类？

“创建类”或“声明类”，是指使用 PHP 的`class`关键字来定义这个蓝图。这个定义明确了此类对象将拥有哪些属性（成员变量）和哪些行为（成员函数或方法）。
# 2 创建类的语法格式
声明一类的基本语法格式如下：
```php
[abstract] [final] class 类名
{
    成员变量定义;
    
    成员函数定义;
}
```
- **`class`**：这是定义类的关键字。
- **`类名`**：为该类指定的标识符（通常建议使用大字母写开头，如`Student`）。
- **`{ ... }`**：花括号内部是一类“封装体”，包含了所有的成员变量和成员函数。
- `[abstract]`/ `[final]`：这是可选的修饰符，用于定义抽象类或最终类。
## 2.1 示例代码
```php
<?php
// 使用 "class" 关键字定义一个名为 "A" 的类
class A
{
    // 1. 成员变量 (属性) 定义
    public $a;
    protected $b;
    private $c;

    // 2. 成员函数 (方法) 定义
    function setA()
    {
        // 在函数内部，使用 $this 访问当前对象的成员
        $this->a = 1; 
        $this->b = 2; 
        $this->c = 3; 
    }
}
?>
```
代码解析：
- `class A { ... }`定义了一个名为`A`的类。
- `public $a;`、`protected $b;`、`private $c;`定义了三个成员变量（属性）。`public`、`protected`、`private`是用于控制访问权限的关键字。
- `function setA() { ... }`定义了一个成员函数（方法）。
