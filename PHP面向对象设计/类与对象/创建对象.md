# 创建对象

在定义了一个类（Class）之后，必须先将其“实例化”，才能使用它。这个“实例”就称为对象（Object）。

# 1 创建对象
创建对象的核心是使用`new`关键字。
## 1.1 无参数构造
如果一个类没有定义构造函数，或者定义了无参数的构造函数，可以使用以下格式创建对象：
```php
对象名 = new 类名;
```
## 1.2 有参数构造
如果一个类定义了带参数的构造函数（`__construct($param1, ...)`），则必须在`new`的同时提供实参表：
```php
对象名 = new 类名(实参表);
```
# 2 对象的创建与访问示例
这里示例展示了如何创建对象，并通过对象访问其公有（`public`）成员。
代码：
```php
<?php
class A
{
    // 定义公有属性
    public $a;
    protected $b;
    private $c;

    // 定义公有方法
    function setA()
    {
        // 在类内部，使用 $this 访问当前对象的属性
        $this->a = 1;
        $this->b = 2;
        $this->c = 3;
    }
}

// 1. 创建对象 (实例化)
// $obj 是类 A 的一个实例（对象）
$obj = new A;

// 2. 通过对象调用方法
$obj->setA();

// 3. 通过对象访问公有属性
// 在类外只能访问 public 成员
echo $obj->a; // 输出: 1
?>
```
# 3 非静态成员与静态成员
在创建对象后，访问其成员（属性和方法）分为两种方式：
## 3.1 非静态成员 (属于对象)
非静态成员属于某个对象实例。不同的对象各自拥有一个非静态成员的副本。。
- 在类内引用：
    - 使用`$`符号。
    - `$this`指代当前对象。
    - `$this->非静态成员`（注意：属性名前面要简洁`$`）。
    - 例如：`$this->a = 1;`
- 类外引用：
    - 使用`->`符号（对象惯例）。
    - `对象名->非静态成员`（注意：属性名前面要简洁`$`）。
    - 例如：`echo $obj->a;`
## 3.2 静态成员（属于类）
静态成员（使用`static`关键字定义）属于整个类，它被该类的所有对象共享 。
- 在类内或类外引用：
    - 使用`::`符号（范围解析符）。
    - `类名::静态成员`（注意：访问静态属性时，属性名前面有`$`）。
- 注意：不能出现在静态方法中 `$this`。
## 3.3 非静态与静态成员的比较
这个例子清楚地展示了非静态属性（`$a`）和静态属性（`$b`）的区别。
代码：
```php
<?php
class Student
{
    var $a;        // 非静态属性 $a
    static $b;   // 静态属性 $b

    // 构造函数
    function __construct()
    {
        $this->a = 20;
        Student::$b = 200; // 在类内访问静态属性
    }
}

// 创建两个 Student 对象
$stul = new Student();
$stu2 = new Student();

// 修改 $stul 的非静态属性 $a
$stul->a = 10;

// 修改 Student 类的静态属性 $b
Student::$b = 100;

// 输出 $stu2 的 $a (非静态，互不影响)
echo '$stu2->a = ' . $stu2->a . '<br>';

// 输出 Student 的 $b (静态，共享)
echo 'Student::$b = ' . Student::$b;
?>

//运行
//$stu2->a = 20 
//Student::$b = 100
```
程序说明： `$stul`和`$stu2`它们有各自的`$a`属性值。修改`$stul->a`不会影响`$stu2->a`。 `$b`属性值为两个对象共享。修改`Student::$b`会影响所有实例（尽管本例中是通过类名直接访问的）。
