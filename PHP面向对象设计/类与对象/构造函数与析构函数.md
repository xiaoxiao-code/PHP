# 构造函数与析构函数

构造函数和解析构造函数是PHP类中两种特殊的成员函数，它们在对象的生命周期中被自动调用。

# 1 构造函数（Constructor）
构造函数的主要作用是在创建对象时执行初始化任务，例如为类的属性赋初始值。
- 声明格式：
```php
[public] function __construct(形参表)
{
    函数体
}
```
注意：
1. 构造函数的名称是固定的：（`__construct`两个下划线初始化）。
2. 一类最多只能定义一个构造函数。
3. 构造函数可以带形参（用于接收创建对象时创建的数据），也可以不带形参。
4. 构造函数在创建对象（使用`new`关键字）时被自动调用 。
# 2 析构函数
解析结构函数的作用主要是在对象被调用之前执行清理任务，例如释放资源。
- 声明格式：
```php
[public] function __destruct()
{
    函数体
}
```
注意：
1. 解析结构函数的名称是固定的：（`__destruct`两个下划线起始）。
2. 一类最多只能有一个解析构造函数。
3. 解析结构函数不带任何形参 。
4. 解析结构函数在释放对象时（例如脚本执行完毕或对象被取消）被自动调用 。
# 3 示例
```php
<?php
class Con
{
    // 构造函数
    function __construct($num)
    {
        echo "执行构造函数 $num <br>";
    }

    // 析构函数
    function __destruct()
    {
        echo "执行析构函数 <br>";
    }
}

$a = new Con("1"); // 创建对象 $a。自动调用 $a 的构造函数
$b = new Con("2"); // 创建对象 $b。自动调用 $b 的构造函数

?>
```
程序解释： 当脚本执行完毕时，对象`$a`并将`$b`被释放，此时会按（与创建时相反的）顺序自动调用它们的构造函数。
