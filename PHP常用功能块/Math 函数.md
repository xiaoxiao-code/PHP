# Math 函数

数学函数主要用于进行数值计算，从简单的取整到复杂的随机数生成。

# 1 基础计算术与取整函数
此类函数用于处理数字的绝对值和取整。

|   函数    |            说明            |                                  效果                                  |
| :-----: | :----------------------: | :------------------------------------------------------------------: |
|  abs()  |   求绝对值。<br>返回一个数的非负值。    |                     echo abs(-4.2);<br>// 输出：4.2                     |
| ceil()  | 向上取整。<br>返回不小于给定值的下一个整数。 |                      echo ceil(4.3);<br>// 输出：5                      |
| floor() | 后续取整。<br>返回不大于给定值的下一个整数。 |                     echo floor(4.8);<br>// 输出：4                      |
| round() |  四舍五入。<br>指定可以保留的小数位数。   | echo round(3.6);<br>// 输出：4<br>echo round(3.14159, 2);<br>// 输出：3.14 |
# 2 幂幂与平方根

| 函数     | 说明                 | 效果                                  |
| ------ | ------------------ | ----------------------------------- |
| pow()  | 侵犯。<br>返回一个数的N次方。  | echo pow(2, 3);<br>// (2的3次方) 输出: 8 |
| sqrt() | 求平方根。<br>返回一个数平方根。 | echo sqrt(16);<br>// 输出：4           |
# 3 随机数生成
生成随机数在开发中非常有用，例如用于生成验证码、抽奖等。

|    函数     |                             说明                             |                     效果                      |
| :-------: | :--------------------------------------------------------: | :-----------------------------------------: |
|  rand()   |                生成随机整数。<br>可以指定一个范围（简单，顶部）。                 |  echo rand(1, 10);<br>// 输出：1到10之间的一个随机整数   |
| mt_rand() | 更好的随机数。<br>使用“Mersenne Twister”算法，它rand()更快，随机性也更好。推​​荐使用。 | echo mt_rand(1, 10);<br>// 输出：1到10之间的一个随机整数 |
# 4 与统计比较

|       函数        |               说明               |                                                              效果                                                              |
| :-------------: | :----------------------------: | :--------------------------------------------------------------------------------------------------------------------------: |
|      max()      |        求顶。返回一组数值中的最大值。         |                                              `echo max(2, 5, 1, 9);`<br>// 输出：9                                              |
|      min()      |       简单地求。返回一组数值中的最小值。        |                                              `echo min(2, 5, 1, 9);`<br>// 输出：1                                              |
| number_format() | 格式化数字（千位分隔符）。这在显示价格、统计数据时非常有用。 | `echo number_format(1234567.89);`<br>// 输出：1,234,568<br>`echo number_format(1234567.89, 2, '.', ',');`<br>// 输出：1,234,567.89 |
# 5 常量
PHP 还提供了一些数学常量。

|  常量  |          说明           |                 效果                  |
| :--: | :-------------------: | :---------------------------------: |
| M_PI | 圆周率 (Pi)。<br>返回波形率的值。 | echo M_PI;<br>// 输出：3.1415926535898 |
