# 其他常用的文件函数

# 1 计算文件大小

函数： `filesize()`
格式：
```php
filesize("文件路径");
```
功能：计算文件的大小，并以字节为单位返回。
# 2 判断文件或目录是否存在
函数： `file_exists()`
```php
file_exists("文件路径/目录路径");
```
功能：检查文件或目录是否存在。如果存在则返回`true`，否则返回`false` 。
# 3 判断是文件还是目录
PHP 还提供了更具体的函数来确定路径的类型：
- `is_dir("路径")`：用于判断指定路径是否是一个目录 。
- `is_file("路径")`：用于判断指定路径是否是一个文件 。
# 4 删除文件
函数： `unlink()`
格式：
```php
unlink("文件路径");
```
功能：删除指定的文件。删除成功则返回`true`，否则返回`false` 。
# 5 复制文件
函数： `copy()`
```php
copy("源文件路径", "目的文件路径");
```
- 功能：将一个文件从源位置复制到新位置。如果复制成功则返回`true`，否则返回`false` 。
- 说明：如果目的文件已存在，将会被覆盖。
复制文件：
```php
<?php
$source = "download/1.txt";
$target = "chap08/download/1.txt";
if (copy($source, $target)) echo "复制成功!";
?>
```
# 6 移动/重命名文件
函数： `rename()`
```php
rename("原文件路径", "新文件路径");
```
功能：将一个文件从原位置移动到新位置，可以保留原名，也可以改名。
移动文件：
```php
<?php
$source = "download/1.txt";
$target = "1.txt";
if (rename($source, $target)) echo "移动成功!";
?>
```
# 7 文件指针操作
当使用`fopen()`打开文件时，PHP 会维护一个指向文件内部位置的“指针”。
- `rewind(文件句柄);`： 重置文件指针，将其移回到文件起点 。
- `ftell(文件句柄);`： 报告文件指针的当前位置（以字节为单位）。
- `fseek(文件句柄, n);`： 将文件指针移动到第`n`字节的位置。
