# UNIX时间戳与转换

在许多UNIX系统中，表示当前日期和时间的常用方法是使用格林尼治时间

# 1 什么是 UNIX 时钟？
- 定义： UNIX计时器是指从1970年1月1日零时零秒（GMT 00:00:00表示）当前时刻所经过的总秒数 。
- 数据类型：它是一个长整数（Long Integer）。
- 跨平台性： Windows系统也可以使用UNIX时钟这使得PHP在处理不同操作系统的数据库时间时，能够有一个统一的时间标准，解决了不同数据库的时间格式兼容问题。
# 2 时间转化为时间戳
在PHP中，人类将日期和时间（如“2016-11-18 23:44:00”）转换为UNIX时钟（一个长整数），主要有两种好的方法：`strtotime()`和`mktime()`。
## 2.1 strtotime() 函数
`strtotime()`函数（String to Time）非常强大，它可以将各种格式的日期时间字符串解析为UNIX时间。
```php
strtotime("Y-m-d H:i:s")
```
```php
<?php
echo strtotime("2016-11-18 23:44:00");
// 输出: 1479479040
?>
```
## 2.2 mktime() 函数
`mktime()`函数通过接收时、分、秒、月、日、年等具体的整数参数来生成一个UNIX时钟。
```php
mktime(H, i, s, m, d, Y)
```
- H: 时（小时）
- i: 分（分钟）
- s：秒（秒）
- m: 月 (Month)
- d：日（日）
- Y：年
```php
<?php
echo mktime(23, 44, 0, 11, 18, 2016);
// 输出: 1479479040
?>
```
