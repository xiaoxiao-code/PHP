# URL参数的格式

URL（统一资源定位符）是Internet上每个网页的地址在动态网页中，URL通常被用来传递少量数据。

# 1 URL参数的格式
```php
url?参数名1=值1&参数名2=值2&参数名3=值3
```
- `?`：分隔符，带有URL路径的结束和参数的开始。
- `&`：分隔符，用于分隔多个参数。
- `参数名=值`：键值对。
# 2 获取参数值的方法
PHP 提供了两个预定义参数来获取 URL 中的参数值（即通过 GET 方式传递的数据）：
1. **`$_GET["参数名"]`**
2. **`$_REQUEST["参数名"]`**
`$_REQUEST`是一个更广泛的变量，它包含`$_GET`和`$_POST`的`$_COOKIE`数据。如果明确知道数据来自 URL，使用`$_GET`是更规范的做法。
# 3 获取超链接参数的示例
这个例子演示了如何在一个页面上点击一个超链接，并提供参数给同一个页面，然后由 PHP 脚本获取这些参数。
```php
<a href="?no=1&name=张三">单击</a>

<?php
// 2. PHP 脚本
// 使用 @ 符号来抑制错误
echo @$_GET["no"];
echo @$_GET["name"];
?>
```
