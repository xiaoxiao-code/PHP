# 获取表单数据的方法

# 1 `$_POST["表单变量"]`

- 用于获取从客户端以POST方式传送过来的表单变量的`value`值。
- `POST`提交的数据方式包含在HTTP请求体中，在浏览器的地址范围不可见，适合提交敏感信息或大量数据。
# 2 `$_GET["表单变量"]`
- 用于获取从客户端以GET方式传递过来的表单变量的`value`值。
- `GET`以 URL 的方式提交数据会附加（例如`?name=test`），在地址栏简短，不适合敏感信息。
# 3 `$_REQUEST["表单变量"]`
- 用于获取从客户端以任何方式（POST 或 GET 等）传递过来的表单变量的`value`值。
- 这是一个便利的标志，表单是`method="post"`还是`method="get"`。
注意：如果表单变量未存在（例如用户未提交表单），访问这些变量将返回`null`。如果表单变量的值为空（例如用户提交了一个空的文本框），则为空字符串`""` 。
# 4 处理不同类型的表单控件
“表单变量”是指表单控件（如文本框、单选按钮等）的`name`属性。
## 4.1 文本框（文本输入）
- HTML： `<input type="text" name="username">`
- PHP： `$user = $_POST["username"];`
## 4.2 单选按钮（单选按钮）
- 规则：一组单选按钮必须设置相同的`name`属性。
- HTML：
```html
<input name="SEX" type="radio" value="男"> 男
<input name="SEX" type="radio" value="女"> 女
```
PHP：
- PHP 脚本用户只需选择该单选按钮的`value`值。
- `$gender = $_POST["SEX"];`(`$gender`的值将为 "男" 或 "女" )。
## 4.3 表单（复选框）
- 规则：如果希望用户能够选择多个组件，它们的`name`属性必须设置相同，并且必须为批量形式（即在`name`属性后加上`[]`）。
- HTML：
```php
<input name="XQ[]" type="checkbox" value="游泳"> 游泳
<input name="XQ[]" type="checkbox" value="看电视"> 看电视
<input name="XQ[]" type="checkbox" value="上网"> 上网
```
PHP：
- PHP 脚本将获取一个数据库，该数据库中保存了用户选择的所有队列的`value`值。
- `$interests = $_POST["XQ"];`（`$interests`将是一个数据库，例如`array("游泳", "上网")`）。
