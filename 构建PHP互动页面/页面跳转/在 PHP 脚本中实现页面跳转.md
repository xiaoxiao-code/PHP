# 在PHP脚本中实现页面跳转

在PHP脚本中，实现页面跳转主要使用`header()`函数。

# 1 语法格式
```php
header("Location: 文件名")
```
# 2 功能说明
- `Location:`是一个标准的HTTP标头字段，它告诉浏览器需要重定向到`文件名`指定的新地址。
- `文件名`可以是一个相对路径（如`EX5-1b.php` ），也可以是一个完整的绝对URL。
```php
// 跳转到一个外部网站
header("Location: http://www.163.com");
```
# 3 重要限制
`header()`函数有一个非常严格的限制：它必须在当前脚本向浏览器发送任何实际输出（例如 HTML 标记、空格或`echo`语句）之前被调用。
