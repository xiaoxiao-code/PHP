# 函数的定义

函数是结构化编程的核心，它允许将一段用于执行特定任务的代码封装起来，并为其命名。
函数的作用是让同一段代码可以被多个地方直接调用，从而节省编程时间、减少错误、易于维护。

# 1 什么是函数定义
所谓函数的定义，就是指采用若干行代码，以指明此函数具体的执行过程、执行前需要输入什么参数，以及执行完成后可以返回什么值 。
# 2 函数定义的语法格式
函数定义的语法格式如下：
```php
function 函数名([形参表])
{
    函数体
}
```
- **`function`**：这是定义函数的关键字。
- **`函数名`**：
    - 这是函数的标识符，用于后续调用。
    - 命名规则：可以是以**字母或下画线**开头，后面跟任意数量的字母、数字、下画线 。
    - 函数名**不能**与系统函数或已经定义过的函数名重名 。
- **`([形参表])`**：
    - 形参（形式参数）列表，是函数从外部接收数据的变量。
    - 形参必须是变量名或数组名 。
    - `[]` 表示这部分是可选的，函数可以没有形参。
- **`{函数体}`**：
    - 由花括号 `{}` 包围的语句块 。
    - 这是函数被调用时实际执行的代码。
# 3 定义的位置
与 C 语言不同的是，PHP 的函数定义**可以随意放在函数调用语句之前或之后** 。PHP 会先读取整个文件的函数定义，再执行调用。
# 4 示例
下面的代码演示了如何定义和调用一个简单的函数 `fun()`。
代码 ：
```php
<?php
// 1. 在定义 *之前* 调用函数
fun(4);

// 2. 函数定义
//    - 函数名: fun
//    - 形参表: $a
//    - 函数体: echo $a*2;
Function fun($a) {
    echo $a * 2;
}

// 3. 在定义 *之后* 调用函数
fun(5);

?>
```
执行结果：
```
8 
10
```
**程序解释：** 第一次调用 `fun(4)` 时，`$a` 得到 4，输出 8。第二次调用 `fun(5)` 时，`$a` 得到 5，输出 10。这证明了函数定义的位置不影响调用。
