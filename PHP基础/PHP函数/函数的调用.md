# 函数的调用

在函数被定义后，必须通过“调用”它来执行其内部的代码。

# 1 fun 函数
```php
Function fun($a) {
    echo $a * 2;
}
```

```php
function fun($x)
{
    if ($x > 0) {
        $y = 1;
    } elseif ($x == 0) {
        $y = 0;
    } else {
        $y = -1;
    }
    
    // 结束函数，并将 $y 的值返回
    return($y); [cite: 1208]
}
```
# 2 函数调用的形式
1. **`函数名(实参表);`**
    - **说明：** 这是最常见的调用形式。它执行函数体内的代码，但不接收函数的返回值。
    - **用途：** 这种形式主要用于调用**无返回值**（或不关心其返回值）的函数 。这类函数的主要目的是执行一个动作，例如 `echo` 输出。
    - **示例：**
```php
fun(5); // 调用 fun 函数，屏幕输出 10 
```
**`变量名 = 函数名(实参表);`**
- **说明：** 此形式执行函数，并将函数 `return` 语句返回的值，赋给等号左侧的变量。
- **用途：** 这种形式用于调用**有返回值**的函数，以便在后续代码中继续使用这个返回的结果 。
- **示例：**
```php
// 调用 fun(-5)，函数返回 -1，并将 -1 赋给 $result 变量
$result = fun(-5); 
echo $result; // 输出 -1
```
**`echo 函数名(实参表);`**
- **说明：** 此形式执行函数，并将函数 `return` 语句返回的值，直接传递给 `echo` 语句进行输出。
- **用途：** 这也是用于调用**有返回值**的函数，是一种将获取返回值和输出值合并的快捷写法 。
- **示例：**
```php
echo fun(10); // 调用 fun(10)，函数返回 1，echo 再将 1 输出 
```
# 3 注意事项
1. **实参与形参 (Arguments vs. Parameters)：**
    - **形参** (Parameters)：是在**定义**函数时声明的变量（例如 `function fun($x)` 里的 `$x`）。
    - **实参** (Arguments)：是在**调用**函数时传入的具体值（例如 `fun(10)` 里的 `10`）。
2. **参数匹配：** 实参的个数、类型必须与形参相同 。
3. **实参的构成：** 实参可以是常量（如 `10`）、变量（如 `$my_num`），也可以是表达式（如 `$a + $b`）。
```php
$a = 5;
$y_val = fun($a * 2); // 合法的调用，传入的实参是表达式 10
```
