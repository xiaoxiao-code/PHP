# 参数的传递

函数参数的传递方式决定了在函数内部对形参（形式参数）的修改是否会影响到函数外部的实参（实际参数）。
PHP 主要支持两种参数传递方式：**值传递**和**引用传递**。

# 1 值传递 (Pass by Value)
这是 PHP **默认**的参数传递方式 。
- **机制：** 函数调用时，是将实参的**值**复制一份传递给形参 。
- **结果：** 在函数内部对形参的值进行任何修改，都**不会**影响到函数外部的实参 。
示例（形参为数组名）：
```php
<?php
// 定义函数，形参 $b 是通过“值传递”
function my_sort($b) 
{ 
    for($i = 0; $i < count($b); $i++)
        $b[$i] += 10; // 尝试修改 $b
}

$a = array(2, 4, 6, 8); // 定义实参 $a
my_sort($a); // 调用函数

// 输出 $a 的值
for($i = 0; $i < count($a); $i++)
    echo $a[$i] . ","; 

// 输出: 2,4,6,8,
// ( $a 的值没有被改变 )
?>
```
**程序解释：** 在 `my_sort($a)` 调用时，PHP 复制了 `$a` 数组并将其赋给 `$b`。函数内部修改的是 `$b`（副本），而原始的 `$a` 保持不变。
# 2 引用传递 (Pass by Reference)
如果希望函数能够修改外部实参的值，就必须使用引用传递。
- **语法：** 在函数定义的**形参**前面加上 `&` 字符 。
- **机制：** 引用传递不再是复制值，而是让形参与实参**共同占用（指向）同一个存储单元** 。
- **结果：** 任何对于形参的值的修改，都**相当于**对实参的值的修改 。
示例 1：形参为变量名
```php
<?php
// 定义函数，形参 $x 是“引用传递”，$y 是“值传递”
function fun(&$x, $y) 
{
    $x += 10;
    $y += 10;
}

$a = 1;
$b = 5;

fun($a, $b); // 调用函数

echo "$a, $b"; // 输出: 11, 5 [cite: 1231]
?>
```
**程序解释：** `$x` 是 `$a` 的引用，因此 `$x += 10` 修改了 `$a`。`$y` 是 `$b` 的副本，因此 `$y += 10` 没有修改 `$b`。
示例 2：形参为数组名
```php
<?php
// 定义函数，形参 $b 是“引用传递”
function my_sort(&$b) 
{ 
    for($i = 0; $i < count($b); $i++)
        $b[$i] += 10; // 修改 $b
}

$a = array(2, 4, 6, 8); // 定义实参 $a
my_sort($a); // 调用函数

// 输出 $a 的值
for($i = 0; $i < count($a); $i++)
    echo $a[$i] . ","; 

// 输出: 12,14,16,18,
// ( $a 的值被改变了 )
?>
```
**程序解释：** `$b` 是 `$a` 的引用，它们指向同一个数组。因此，对 `$b` 中元素的修改就是对 `$a` 中元素的修改。
