# 函数的返回值

在PHP中，函数不仅可以执行操作（如输出文本），还可以“返回”一个值给调用它的主调语句。
这是通过 `return` 语句实现的。

# 1 `return` 语句
`return` 语句有两个主要作用 ：
1. **结束函数的运行**：一旦PHP执行到 `return` 语句，该函数会立即停止执行，`return` 之后的任何代码都将被忽略。
2. **返回一个值**：`return` 会将一个值发送回调用该函数的地方（主调语句）。
# 2 语法格式
`return` 语句的语法格式如下 ：
```php
return [(表达式)];
```
- **功能**：将 `(表达式)` 的计算结果（作为函数值）带回主调函数中 。
- `[]` 方括号表示 `(表达式)` 是可选的。如果省略表达式（即 `return;`），函数会返回一个 `NULL` 值。
# 3 带返回值的函数
下面的示例定义了一个函数 `fun()`，它根据输入的 `x` 值返回 1、0 或 -1 。
程序代码 ：
```php
<?php
// 函数定义
function fun($x)
{
    if ($x > 0) {
        $y = 1;
    } elseif ($x == 0) {
        $y = 0;
    } else {
        $y = -1;
    }
    
    // 结束函数，并将 $y 的值返回
    return($y); [cite: 1208]
}

// --- 主调语句 ---
// 调用 fun() 函数，并将返回值传递给 echo 语句
echo fun(10);

?>
```
**执行流程：**
1. `echo fun(10);` 开始执行。
2. `fun(10)` 被调用，`$x` 的值为 10。
3. 函数体内部，`if ($x > 0)` 条件为真，`$y` 被赋值为 `1`。
4. `return($y);` 语句被执行，函数立即停止，并将 `$y` 的值（即 `1`）发送回去。
5. `echo fun(10);` 中的 `fun(10)` 部分现在被替换为它返回的值 `1`。
6. 最终，`echo 1;` 被执行，页面输出 `1`。
**`return` (返回值) 与 `echo` (输出) 的区别：**
- **`echo`**：只是将内容直接发送到浏览器显示。
- **`return`**：是将值发送回调用它的PHP代码，以便该值可以被赋给变量或用于进一步的计算
