# 外部变量

“外部变量”特指那些从客户端传递到服务器的数据名称。
具体来说，外部变量主要包括 ：

1. **表单中的变量**（即HTML控件的 `name` 属性名）。
2. **URL 中的参数名**（即 `?` 后面 `key=value` 中的 `key`）。
# 1 如何访问外部变量
PHP 不会将这些外部变量自动创建为 `$` 变量。相反，必须通过特定的预定义变量（超全局数组）来获取它们的值 。
主要使用以下三个预定义变量：
- **`$_POST["表单变量"]`** 用于取得从客户端以 **POST** 方式传递过来的表单变量的 `value` 值 。
    - **场景：** `<form method="post">` 提交的数据。
- **`$_GET["表单变量"]`** 用于取得从客户端以 **GET** 方式传递过来的表单变量的 `value` 值 。
    - **场景：** `<form method="get">` 提交的数据，或 URL 中的参数（如 `page.php?name=test`）。
- **`$_REQUEST["表单变量"]`** 用于取得从客户端以**任意方式**（POST, GET, COOKIE等）传递过来的表单变量的 `value` 值 。
- **`$_REQUEST["参数名"]`** 同样，此变量也可以用于取得从客户端（如 URL 中）传递过来的参数值 。
# 2 示例
## 2.1 HTML 表单 (POST 方式)
```html
<form name="form1" method="post" action="">
    用POST发送的学号:
    <input type="text" name="XH" />
    <input type="submit" name="button1" value="提交" />
</form>
```
## 2.2 PHP 接收 (使用 `$_POST`)
```php
<?php
if(isset($_POST['button1'])) {
    // "XH" 就是外部变量名
    // 通过 $_POST['XH'] 来获取它的值
    echo '学号:' . $_POST['XH'];
}
?>
```
