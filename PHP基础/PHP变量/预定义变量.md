# 预定义变量

PHP 预定义了许多特殊的变量，它们在脚本的任何作用域中都自动有效，无需使用 `global` 关键字。这些变量被称为“预定义变量”或“超全局变量”。
它们通常是以数组的形式存在，由PHP预先设置，用于存储来自服务器、运行环境、用户输入等方面的数据 。

# 1 `$GLOBALS`
`$GLOBALS` 是一个特殊的数组，它**以数组形式记录了所有已经定义的全局变量** 。
- **功能：** 允许在任何作用域（例如函数内部）直接访问或修改主程序中定义的全局变量 。
- **与 `global` 关键字的对比：** 它比使用 `global $var;` 语句更方便，因为它将所有全局变量集中在一个地方 。
示例 ：
```php
<?php
$a = 1;
$b = 2;

function fun() {
    // 使用 $GLOBALS 访问和修改全局变量 $b
    $GLOBALS['b'] = $GLOBALS['b'] + 10; 
}

fun();
echo "$a, $b"; // 输出: 1, 12 (变量 $b 的值被函数修改了)
?>
```
# 2 `$_SERVER`
`$_SERVER` 是一个由 Web 服务器创建的数组，它包含了服务器信息、执行环境信息、文件头信息、路径和脚本位置等 。
常用成员 (部分示例) ：
- **`$_SERVER['PHP_SELF']`：** 当前正在执行脚本的文件名（相对于文档根目录） 。
- **`$_SERVER['SERVER_ADDR']`：** 服务器的IP地址 。
- **`$_SERVER['DOCUMENT_ROOT']`：** 当前脚本所在的文档根目录 。
- **`$_SERVER['HTTP_HOST']`：
	- **`Host` 头信息，通常是访问的域名或 `localhost` 。
- **`$_SERVER['REMOTE_ADDR']`：** 当前浏览网页的用户的IP地址 。
# 3 `$_POST`
`$_POST` 是一个数组，用于收集通过 **POST** 方法从客户端（通常是HTML表单）传递过来的数据 。这是在同页和跨页交互中最常用的变量之一。
**示例：**
- HTML表单：`<input type="text" name="Rad">`
- PHP接收：`$Rad = $_POST['Rad'];`
# 4 `$_GET`
`$_GET` 是一个数组，用于收集通过 **GET** 方法从客户端传递过来的数据 。GET 数据通常显示在 URL 的查询字符串中（即 `?` 之后的部分）。
**示例：**
- URL: `http://example.com/page.php?id=123`
- PHP接收：`$id = $_GET['id'];` ( `$id` 的值将是 "123" )
# 5 `$_REQUEST`
`$_REQUEST` 是一个数组，它默认包含了 `$_GET`、`$_POST` 和 `$_COOKIE` 的内容。它可以取得从客户端以任意方式传递过来的表单变量的值 。
- **便利性：** 无需关心数据是通过 `GET` 还是 `POST` 提交的。
- **注意：** 在某些情况下，为了更明确和安全，推荐分别使用 `$_GET` 和 `$_POST`。
# 6 `$_ENV`
`$_ENV` 是一个数组，用于存储PHP运行环境相关的信息 。它包含由服务器操作系统传递给PHP的环境变量，例如系统路径（Path）、操作系统（OS）等 。
# 7 其他预定义变量
- **`$_SESSION`：** 用于存储会话数据。
- **`$_COOKIE`：** 用于访问 `Cookie` 数据。
- **`$_FILES`：** 用于处理文件上传。
