# PHP 数据类型

# 1 整型 (Integer)

整型（int）用于表示没有小数部分的整数。
- **表示方式：** 整型常量可以用十进制、十六进制（`0x` 开头）、八进制（`0` 开头）或二进制（`0b` 开头）来表示 。
- **示例：**
```php
$decimal_num = 100;
$hex_num     = 0x12;  // 十六进制, 等于十进制的 18 
$octal_num   = 012;   // 八进制, 等于十进制的 10 
$binary_num  = 0b1000; // 二进制, 等于十进制的 8
```
# 2 浮点型 (Float / Double)
浮点型（float），也称为 `double`，用于表示带小数的数字或指数形式的数字。
- **表示方式：** 浮点型常量有两种表示法：
    1. **小数形式：** 如 `0.0528`
    2. **指数形式：** 如 `5.28e-2` (表示 5.28 乘以 10 的 -2 次方)
- **示例：**
```php
$pi = 3.14159;
$scientific = 5.28e-2;
```
# 3 字符串型 (String)
字符串型（string）是一系列字符的序列，用于表示文本。PHP中的字符串必须用单引号（`'`）或双引号（`"`）括起来 。
## 3.1 单引号 (`'...'`)
单引号字符串按**字面意义**处理。
- **规则：** 单引号内的变量名和大多数转义序列（如 `\n`）**不会**被解析。
- **例外：**
    - 要表示一个单引号本身，必须使用反斜杠转义：`\'` 。
    - 要表示一个反斜杠本身，必须使用反斜杠转义：`\\` 。
- **示例 ：**
```php
$str_var = "加油";
echo '这个字符串有\n，但不转义'; // 输出: 这个字符串有\n，但不转义
echo '中国$str_var!';             // 输出: 中国$str_var!
```
## 3.2 双引号 (`"..."`)
双引号字符串会**解析**其中的内容。
- **规则：**
	- **变量解析：** 双引号内的变量名（如 `$str_var`）会被替换为其存储的值 。
	- **转义序列：** 特殊的转义序列会被解析为其代表的含义，例如 `\n`（换行）、`\t`（制表符）等 。
- **示例 ：**
```php
$str_var = "加油";
echo "这个字符串有\n，会转义"; // 输出: 这个字符串有
								//  ，会转义
echo "中国$str_var!";             // 输出: 中国加油!
```
# 4 布尔型 (Boolean)
布尔型（boolean）是逻辑类型，只有两种取值，用于表示真伪状态。
- **取值：** `TRUE` 或 `FALSE` (不区分大小写) 。
- **显示：** 当使用 `echo` 输出一个布尔值时，`TRUE` 会显示为 `1`，而 `FALSE` 显示为空字符串（即什么也看不见） 。
- **示例：**
```php
$is_active = true;
$is_logged_in = false;
```
# 5 数组 (Array)
数组（array）是一种复合数据类型，它允许在一个变量中存储一组“键名/值”对（Key/Value pairs） 。
- **键名 (Key)：** 数组中每个值的唯一标识符。键名可以是**整数**或**字符串** 。
- **值 (Value)：** 键名所对应的数据，可以是任何PHP数据类型。
- **规则：**
    - 如果创建数组时省略了键名，PHP会默认使用从 `0` 开始的连续整数作为键名 。
- **示例：**
```php
// 1. 默认整数键名 (0, 1, 2...)
$arr1 = array(2, 4, 6, 8);
// 访问: echo $arr1[0]; // 输出 2

// 2. 自定义字符串键名
$arr2 = array('name' => "Li", 'age' => 20, 'married' => false);
// 访问: echo $arr2['name']; // 输出 Li 
```
# 6 对象 (Object)
对象（object）是面向对象编程（OOP）的概念。它是一种复合数据类型，通常用于将数据（称为属性）和操作这些数据的函数（称为方法）封装在一起。
- **来源：** 对象是通过 `new` 关键字实例化一个 `class`（类）而创建的。
- **说明：** 这是一个高级数据类型。
- **示例：**
```php
// (假设已定义了一个 Student 类)
$student1 = new Student();
```
# 7 NULL 类型
NULL 是一种特殊的数据类型，它只有一个值：`NULL`（不区分大小写） 。
- **含义：** `NULL` 表示一个变量**没有值**。
- 一个变量在以下情况下被认为是 `NULL` ：
    1. 被显式赋值为 `NULL` 。
    2. 尚未被赋予任何值 。
    3. 被 `unset()` 函数销毁 。
- **示例：**
```php
$var1 = NULL; // 1. 显式赋值
$var2;        // 2. 尚未赋值

$var3 = "test";
unset($var3); // 3. 被unset()销毁
```
# 8 资源类型 (Resource)
资源（resource）是一种特殊的变量，它不是具体的数据，而是**一个指向外部资源的引用** 。
- **含义：** 它相当于一个“句柄”或“指针”，由PHP的特定函数（通常是用于操作数据库、文件、网络连接等的函数）创建和返回。
- **示例：**
    - 最常见的例子是**数据库连接** 。
    - 其他例子包括打开的文件句柄。
- **示例代码：**
```php
$db_connection = mysql_connect("localhost", "root", "password");
// 此时 $db_connection 变量就是一个 "resource" 类型。
```
