# 循环控制语句

# 1 while 语句

`while` 语句的特点是**先判断后执行** 。如果条件一开始就为假，循环体可能一次也不执行。
语法格式 ：
```php
while (表达式)
    语句
```
- **逻辑：** 只要括号中的 `表达式` 求值为 `true`，就重复执行其后的 `语句`（或语句块 `{}`）。
用 while 语句求 1~100 的偶数和 ：
```php
<?php
$s = 0;
$i = 2; // 循环变量赋初值

while ($i <= 100) { // 循环条件
    $s = $s + $i;
    $i = $i + 2; // 循环变量更新
}

echo "和为:" . $s;
?>
```
# 2 do-while 语句
`do-while` 语句的特点是**先执行后判断** 。这保证了循环体**至少会执行一次**。
语法格式 ：
```php
do
    循环体语句
while (表达式);
```
- **注意：** `while(表达式)` 之后的分号（`;`）是必需的 。
用 do-while 语句求 1~100 的偶数和：
```php
<?php
$s = 0;
$i = 2;

do {
    $s = $s + $i;
    $i = $i + 2;
} while ($i <= 100);

echo "和为:" . $s;
?>
```
# 3 for 语句
`for` 语句是结构最紧凑的循环语句，它将循环的三个关键部分（初始化、条件、更新）集中在一行。
语法格式 ：
```php
for (表达式1; 表达式2; 表达式3)
    循环体语句
```
- 执行过程 ：
    1. **表达式1：** 循环变量赋初值。仅在循环开始前执行一次。
    2. **表达式2：** 循环条件。在每次循环体执行**之前**进行判断，若为 `true` 则执行循环体，若为 `false` 则退出循环。
    3. **循环体语句：** 执行循环的主体代码。
    4. **表达式3：** 循环变量更新。在每次循环体执行**之后**执行。
    5. 返回第 2 步。
- **说明：** 这三个表达式都可以省略，但分号（`;`）不能省略 。`for(;;)` 是一种“死循环”。
用 for 语句求 1~100 的偶数和 ：
```php
<?php
$s = 0;

for ($i = 2; $i <= 100; $i += 2) {
    $s = $s + $i;
}

echo "和为:" . $s;
?>
```
# 4 foreach 语句
`foreach` 语句是专门用于**遍历数组**的循环语句，它比 `for` 循环遍历数组更简洁。
语法格式 ：
```php
foreach (数组名 as $成员名)
    语句
```
**功能：** 对于数组中的每个元素，都重复执行一次循环体。在每次循环中，当前数组元素的值会被赋给 `$成员名` 变量。
示例：
```php
<?php
// 定义一个数组
$a = array(3, 6, 9, 12, 15, 18);

// 遍历数组 $a
foreach ($a as $x) {
    // $x 在每次循环中会依次变为 3, 6, 9 ...
    echo $x . ", ";
}

// 输出: 3, 6, 9, 12, 15, 18, 
?>
```
