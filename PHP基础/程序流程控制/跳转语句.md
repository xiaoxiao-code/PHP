# 跳转语句

# 1 break 语句

`break` 语句用于**完全中止**并**跳出**当前的 `switch` 结构或循环（`for`, `while`, `do-while`, `foreach`）。
- **用途：**
    - 在 `switch` 语句中，`break` 用于防止“穿透”，即执行完一个 `case` 后立即退出 `switch` 块。
    - 在循环体内，`break` 通常与 `if` 语句配合使用，当满足某个特定条件时，提前终止整个循环 。
- **示例（在循环中使用）：**
```php
// 查找 1 到 100 之间第一个能被 7 整除的数
for ($i = 1; $i <= 100; $i++) {
    if ($i % 7 == 0) {
        echo "第一个能被7整除的数是: " . $i;
        break; // 找到后，立即跳出 for 循环
    }
}
// 循环会在此处停止，不会继续检查 14, 21...
```
# 2 continue 语句
`continue` 语句用于**终止当前的循环迭代**，并立即**重新开始下一次循环** 。
- **用途：**
    - `continue` 只能用于循环体内，它不会跳出整个循环，而是跳过当前迭代中 `continue` 语句之后的所有代码 。
    - 它也常与 `if` 语句配合使用，用于在特定条件下跳过当前的数据处理，直接进入下一轮 。
- **示例（在循环中使用）：**
```php
// 输出 1 到 10 之间所有的奇数
for ($i = 1; $i <= 10; $i++) {
    if ($i % 2 == 0) {
        // 如果 $i 是偶数
        continue; // 跳过本次循环的剩余部分（echo语句）
    }
    // $i 是奇数时，这行代码才会被执行
    echo $i . ", ";
}
// 输出: 1, 3, 5, 7, 9,
```
