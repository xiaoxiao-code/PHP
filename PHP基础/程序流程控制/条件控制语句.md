# 条件控制语句

# 1 if 语句

`if` 语句是最基本的条件控制。
## 1.1 基本 `if` 和 `if-else` 结构
`if` 语句用于解决单分支或双分支问题 。
语法格式 ：
```php
if (表达式)
    语句1;
[else
    语句2;]
```
- **逻辑：**
    1. 首先，对括号中的 `表达式` 进行求值。
    2. 如果表达式的值为 `true`（真），则执行 `语句1` 。
    3. 如果表达式的值为 `false`（假），则跳过 `语句1`。
    4. 如果存在 `else` 子句，则在表达式为 `false` 时执行 `语句2` 。
- **语句块：** `语句1` 和 `语句2` 可以是单个语句（以分号 `;` 结尾），也可以是花括号 `{}` 括起来的语句块。
示例（双分支）：
```php
if ($a > $b) {
    $max = $a;
} else {
    $max = $b;
}
```
## 1.2 `if-elseif-else` 结构 (嵌套)
当存在多个分支条件时，使用 `if` 语句的嵌套格式 。
语法格式 ：
```php
if (表达式1)
    语句1;
elseif (表达式2)
    语句2;
elseif (表达式3)
    语句3;
...
else
    语句n;
```
- **逻辑：** 程序会从上到下依次判断表达式。一旦遇到为 `true` 的表达式，就执行其对应的语句，然后**跳过**所有后续的 `elseif` 和 `else`。如果所有表达式都为 `false`，则执行 `else` 部分的语句（如果存在）。
- **注意：** 在PHP中，`elseif` 也可以分开写成 `else if` 。
成绩等级转换：
```php
<?php
if (isset($_REQUEST["button"])) {
    $score = $_REQUEST["score"];
    
    if ($score >= 90) {
        $grade = 'A';
    } elseif ($score >= 80) { // 隐含了 $score < 90
        $grade = 'B';
    } elseif ($score >= 70) { // 隐含了 $score < 80
        $grade = 'C';
    } elseif ($score >= 60) { // 隐含了 $score < 70
        $grade = 'D';
    } else {
        $grade = 'E';
    }
    
    echo "成绩等级为: " . $grade;
}
?>
```
# 2 switch 语句
`switch` 语句也称为多分支语句，它适用于根据一个**表达式的固定值**来决定执行哪个分支 。
语法格式 ：
```php
switch (表达式) {
    case 常量1:
        语句块1;
        [break;]
    case 常量2:
        语句块2;
        [break;]
    ...
    [default:
        语句块n+1;]
}
```
- **逻辑：**
    1. 首先，计算 `switch` 后面 `表达式` 的值 。
    2. 程序将这个值与每个 `case` 后的 `常量` 逐一比较。
    3. 如果找到相等的值，就执行该 `case` 块中的语句 。
    4. 如果所有 `case` 都不匹配，则执行 `default` 块中的语句（如果存在）。
- **`break` 语句：** `break` 语句用于跳出 `switch` 结构。如果省略 `break`，程序会继续执行**下一个 `case`** 中的代码，这称为“穿透”。
- **注意事项：**
    1. `switch` 中的 `表达式` 类型可以是数值型或字符串型 。
    2. 多个不同的 `case` 可以执行同一个语句块（通过省略 `break` 实现）。
    判断月份天数 ：
```php
<?php
if (isset($_REQUEST["button"])) {
    $month = $_REQUEST["month"];
    
    switch ($month) {
        case 1:
        case 3:
        case 5:
        case 7:
        case 8:
        case 10:
        case 12:
            $day = 31;
            break;
        case 4:
        case 6:
        case 9:
        case 11:
            $day = 30;
            break;
        case 2:
            $day = 28; // (暂不考虑闰年)
            break;
        default:
            $day = 0; // 表示无效月份
    }
    
    echo $month . "月份的天数为: " . $day;
}
?>
```
