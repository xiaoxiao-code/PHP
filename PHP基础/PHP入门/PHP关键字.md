# PHP关键字

关键字是PHP语言内部预先定义和保留的单词，它们具有特殊的含义，用于构成语言的语法。
**规则：** 关键字不能被用作标识符（如函数名、类名、常量名）。例如，`function if()` 或 `class echo` 这样的定义是错误的。
以下是PHP中常用的一些关键字总结

|  类别   |            关键字             |           描述           |
| :---: | :------------------------: | :--------------------: |
| 流程控制  |      if, else, elseif      |       用于条件判断分支。        |
| 流程控制  |   switch, case, default    |       用于多分支选择结构。       |
| 流程控制  |         while, do          | while 和 do-while 循环语句。 |
| 流程控制  |            for             |       for 循环语句。        |
| 流程控制  |          foreach           |        用于遍历数组。         |
| 流程控制  |           break            |    用于跳出 switch 或循环。    |
| 流程控制  |          continue          |     用于跳过当前循环的剩余部分。     |
| 函数与类  |          function          |       用于定义一个函数。        |
| 函数与类  |           return           |        从函数中返回值。        |
| 函数与类  |           class            |        用于定义一个类。        |
| 函数与类  |            new             |    用于创建类的一个新实例（对象）。    |
| 函数与类  |          extends           |    用于类的继承（子类继承父类）。     |
| 函数与类  | public, protected, private | 定义类成员的访问权限（公有、保护、私有）。  |
| 函数与类  |           static           |  定义静态成员（属于类本身，而非对象）。   |
| 函数与类  |          abstract          |     用于定义抽象类或抽象方法。      |
| 函数与类  |         interface          |        用于定义接口。         |
| 函数与类  |         implements         |      用于实现一个或多个接口。      |
| 函数与类  |           const            |       用于声明一个常量。        |
| 输出与包含 |            echo            |    向浏览器输出一个或多个字符串。     |
| 输出与包含 |           print            |  类似于 echo，用于输出一个字符串。   |
| 输出与包含 |      include, require      |    在当前脚本中包含并执行指定文件。    |
| 输出与包含 | include_once, require_once |      确保文件只被包含一次。       |
| 变量与值  |           global           |      在函数内部访问全局变量。      |
| 变量与值  |           unset            |        销毁一个变量。         |
| 变量与值  |        true, false         |     布尔值的两种状态（真和假）。     |
| 变量与值  |            null            |    表示一个变量没有值的特殊类型。     |
