# PHP 注释

注释是PHP脚本中用于解释和说明代码的文本，它们会被PHP解释器完全忽略，不会被执行 。注释对于代码的维护和团队协作至关重要。
PHP支持两种注释风格，这与C语言和Java类似。

# 1 单行注释 (`//`)
- **语法：** 以 `//` 开始，直到该行的末尾，所有内容均被视为注释 。
- **用途：** 适用于对单行代码进行简短说明。
示例：
```php
<?php
echo "Hello"; // 输出 Hello [cite: 636]
echo "World";
// 下面这行代码不会被执行
// echo "Goodbye";
?>
```
# 2 多行注释 (`/* ... */`)
- **语法：** 以 `/*` 开始，以 `*/` 结束 。
- **用途：** 适用于跨越多行的详细说明，或者临时“注释掉”一大段代码块以便调试。
示例 ：
```php
<?php
/*
  这是一个多行注释块。
  下面的代码将执行两个echo语句。
  [cite: 639, 641]
*/
echo "Hello";
echo "World";
?>
```
