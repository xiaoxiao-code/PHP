# PHP标记符

PHP标记符是用于告知服务器（Apache）哪部分代码需要被PHP解释器执行的特殊定界符。
在一个PHP文件（`.php`）中，可以混合编写HTML标记和PHP脚本。服务器在处理文件时，会将标记符**之外**的内容视为普通HTML文本直接发送给浏览器，而**只执行**位于标记符**之内**的代码 。

# 1 标准PHP标记
PHP脚本的标准标记（也是本书中唯一使用的标记）是：
- **开始标记：** `<?php`
- **结束标记：** `?>`
**代码：**
```php
<!DOCTYPE html>
<html>
<body>
    
    <p>这是一段HTML文本，将直接发送到浏览器。</p>
    
    <?php
    // 这部分是PHP代码，将在服务器上执行
    echo '<p>这段文本是由PHP生成的。</p>';
    ?>
    
    <p>这是另一段HTML文本。</p>

</body>
</html>
```
- **说明：** PHP允许在一个文件中存在多对开始和结束标记 。
- **注意：** 虽然存在其他样式的标记（如短标记 `<?`），但它们在所有服务器上的支持并不统一。因此，始终使用 `<?php` 是最规范、兼容性最好的做法 。
