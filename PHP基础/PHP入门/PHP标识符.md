# PHP 标识符

标识符是程序员在代码中为特定编程元素（如变量、函数、类）所指定的名称。

# 1 标识符的适用范围
标识符用于命名：
- **变量：** 例如 `$name`, `$user_age`
- **函数：** 例如 `calculateArea()`, `connect_db()`
- **常量：** 例如 `MAX_SIZE`
- **类：** 例如 `Student`, `DatabaseHelper`
- **类的方法（函数）与属性（变量）**
# 2 标识符的命名规则
PHP的标识符（不包括变量）必须遵循以下规则：
- **首字符：** 必须以字母（`a-z`, `A-Z`）或下划线（`_`）开头 。
- **后续字符：** 可以是字母、数字（`0-9`）或下划线（`_`）的任意组合 。
- **禁止项：** 不能以数字开头，名称中不能包含空格或除下划线外的特殊字符。
- **关键字：** 标识符的名称**不能**是PHP的关键字。
**变量的特殊规则：** PHP 变量自成一派，它们必须以美元符号（`$`）开头 。`$` 符号后面的名称遵循上述通用标识符规则（例如 `$_myVar`, `$user2`） 。
# 3 标识符的大小写敏感性
这是一个非常重要的特性，必须严格区分：
- **大小写敏感 (Case-Sensitive)：**
    - **变量**是严格区分大小写的。
    - 例如：`$name`、`$Name` 和 `$NAME` 是三个**不同**的变量。
- **大小写不敏感 (Case-Insensitive)：**
    - **函数名**、**类名** 和 **关键字** 是不区分大小写的。
    - 例如：调用 `myFunction()`、`myfunction()` 或 `MYFUNCTION()` 都是在调用同一个函数。
