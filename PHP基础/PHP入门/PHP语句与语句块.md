# PHP语句与语句块

# 1 PHP 语句

在PHP中，一个“语句”是执行特定操作的单一指令。可以将其理解为一个完整的“句子”或“命令”。
**关键规则：** 在PHP中，绝大多数语句**必须**以分号（`;`）结尾。分号是PHP用来区分不同语句的定界符 。
**语句的示例：**
1. **赋值语句：**
```php
$a = 100;
$b = "Hello, World!";
```
2. **输出语句：**
```php
echo '这是一个语句'; 
print '这是另一个语句';
```
3. **函数调用语句：**
```php
session_start(); 
$area = $Rad * $Rad; 
```
4. **流程控制语句：**
```php
break; 
continue;
return $y; 
```
# 2 PHP 语句块
“语句块”是将**零个或多个语句**组合在一起，使其在语法上被视为一个单一单元。
**关键规则：** 语句块由一对花括号（`{ ... }`）界定。
语句块本身不是一个完整的命令，它必须依附于某个控制结构（如 `if`、`while`、`function` 等）来使用。
**语句块的应用场景：**
1. **条件语句 (`if`, `elseif`, `else`):**
    `if` 语句检查一个条件。如果该条件为真，则执行其后的语句块 。
```php
if (isset($_POST['send'])) {
    // --- 语句块开始 ---
    $Rad = $_POST['Rad'];
    $area = $Rad * $Rad;
    echo '正方形的面积是' . $area;
    // --- 语句块结束 ---
}
```
2. **循环语句 (`while`, `do-while`, `for`, `foreach`):** 循环语句会重复执行其后的语句块 。
```php
while ($i <= 100) {
    // --- 语句块开始 ---
    $s = $s + $i;
    $i = $i + 2;
    // --- 语句块结束 ---
}
```
3. **函数定义 (`function`):** 函数的主体（即函数被调用时要执行的代码）必须是一个语句块 。
```php
function show($message) {
    // --- 语句块开始 ---
    echo "<script>alert('$message')</script>";
    // --- 语句块结束 ---
}
```
4. **`switch` 语句：**
`switch` 语句的主体也是一个语句块 。
# 3 语句与语句块的关系
这是初学者容易混淆的地方：**控制结构（如 `if`, `while`）在语法上只控制其后的一个单元。**
- **情况一 (使用语句块 - 推荐)：** 如果使用花括号 `{}`，那么 `if` 会控制花括号内的**所有语句**。
```php
if ($a > $b) {
    echo "A 大于 B";
    $max = $a; // 此行受 if 控制
}
```
- **情况二 (使用单个语句 - 不推荐)：** 如果**省略**花括号 `{}`，那么 `if` **只控制**其后的**第一个语句**（直到第一个分号 `;`）。
```php
if ($a > $b)
    echo "A 大于 B"; // 此行受 if 控制

$max = $a; // 错误：此行 *不受* if 控制，它将永远执行！
```
